Servlet?

  1. Servlet은 클라이언트의 요청을 분석하여 적절한 응답 서비스를 제공하는 자바 클래스입니다.
  2. Servlet은 서버에서 수행하는 기능을 구현하는 프로그램입니다.

웹서버에서 동작하는 Servlet 클래스를 생성할 때는 주로 'javax.servlet.http.HttpServlet'을 상속하여 구현합니다.

HttpServlet 메서드
  1. init() : Servlet 클래스의 객체가 생성된 후에 호출되는 메서드
  2. destroy() : Servlet 객체가 메모리에서 해제될 때 호출되는 메서드
  3. service() : 요청 방식에 관계없이 브라우저로부터 요청이 전달되면 호출되는 메서드
  4. doGet() : 브라우저로부터 GET 방식으로 요청이 전달되면 호출되는 메서드
  5. doPost() : 브라우저로부터 POST 방식으로 요청이 전달되면 호출되는 메서드
