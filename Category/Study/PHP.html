PHP?

예로부터 조상님들께서는 지피지기면 백전백승이라말씀하시지 않았던가..
php라는 언어의 활용을 본격적으로 공부하기 전에 php가 무엇이며 어떤 역할을 하는 확인하고 넘어가자.
그러고보니... '지피지기면 백전백승' 너무 오랜만에 사용한다...
(다음의 내용은 'https://www.w3schools.com/php/php_intro.asp'를 참고하였습니다.)

  1. php는 서버측에서 작동하는 스크립트언어입니다.
  2. php 파일(.php)은 텍스트와 HTML, CSS, JavaScript, PHP 코드를 수용할 수 있습니다.
  3. php 코드는 서버에서 작동하며 작업의 결과를 순수한 HTML로 브라우저에게 리턴합니다.
  4. php 동적페이지 컨텐츠를 생성할 수 있습니다.
  5. php는 서버에 있는 파일을 생성하고 열고, 읽고, 쓰고, 삭제하고, 닫을 수 있습니다.
  6. php는 양식데이터를 수집할 수 있습니다.
  7. php는 쿠키를 보내고 받을 수 있습니다.
  8. php는 데이터베이스에 있는 데이터를 추가하고, 삭제하고, 수정할 수 있습니다.
  9. php는 유저의 접근을 컨트롤하는것에 사용될 수 있습니다.
  10. php는 데이터를 암호화 할 수 있습니다.

Mechanism of PHP

자.. 앞에서 적이 어떤 녀석인지 적당히 알아 본 것 같다.
w3s에서 언급한 내용만 보면 Spring으로 웹을 구현할 때 사용했던 jsp와 servlet의 역할을 동시에 하고 있는 것 같다는 생각이든다. 아니면 jsp의 역할만 하고 있나?
궁금한 사항은 차차 학습하며 알게 될 것이고... 다음은 php가 클라이언트의 web browser와 server computer의 관계에서 어떻게 동작하는지 php의 작동 원리를 알아보자.
다음의 내용은 유튜브 생활코딩의 이고잉 선생님이 함께해주셨다. 선생님은 앞으로도 많이 함께해주실 예정.. 감사합니다..

클라이언트가 서버 컴퓨터에 '.html'의 확장자를 가진 요청을 하였을 경우
  1. 클라이언트의 웹 브라우저가 서버컴퓨터에 'index.html'의 요청을 합니다.
  2. 서버컴퓨터의 웹 서버 소프트웨어는 '.html'의 확장자를 확인하고, 웹 서버 스스로 처리할 수 있는 요청임을 인지합니다.
  3. 웹 서버는 서버컴퓨터의 저장소에 설치되어있는 htdocs에서 index.html파일을 읽어 클라이언트에게 전송합니다.
  4. 클라이언트의 웹 브라우저는 전송받은 html을 해석해서 화면에 출력합니다.
  5. '.html'의 데이터는 정적입니다.
클라이언트가 서버컴퓨터에 '.php'의 확장자를 가진 요청을 하였을 경우
  1. 클라이언트의 웹 브라우저가 서버 컴퓨터에 'index.php'의 요청을합니다.
  2. 웹 서버는 '.php'의 확장자를 확인하고 스스로 처리할 수 없는 요청임을 인지합니다.
  3. 웹 서버는 '.php' 요청의 처리를 PHP 프로그램에 위임합니다.
  4. PHP는 htdocs라는 디렉토리 안에 있는 'index.php' 파일을 확인합니다.
  5. 해당 파일에 적혀있는 php 언어를 해석하고 최종적으로 '.html' 파일을 생산합니다.
  6. PHP에 의해 생산된 ',html'을 웹 서버가 클라이언트에게 전송합니다.
  7. 클라이언트의 웹 브라우저는 전송받은 html을 해석해서 화면에 출력합니다.
  8. '.php'의 데이터는 '' 안에 적혀있는 php 문법에 따라 실시간으로 html을 생성하므로 동적입니다.

Data Type of PHP

php의 기본적인 개념은 이쯤하고 이제 구체적인 문법을 공부할 때가 되었다.
프로그래밍 언어를 사용하는 이유 중에 하나는 데이터를 어떠한 의도에 따라 처리하기 위함일 것이다.
이러한 이유로 인해 '프로그래밍 언어가 어떠한 데이터 타입을 제공하는가 그리고 각각의 데이터 타입을 처리하는 방법을 어떻게 제공하는가'에 대해 공부하는 것은 필수적이다.

'php.net'의 php manual에 따르면 php는 10개의 기본 데이터 타입을 제공한다.
그중에 4개는 scalar 타입이며 bool, int, float(floating-point numbre, aka double), string이 scalar타입에 해당한다.
(네이버 사전에 따르면 'scalar'가 '수치(또는 실수)로 표시되는 수량'의 의미를 가지고 있다고 하는데... string은 왜 scalar 타입인거지?)
또 다른 4개는 compound 타입이고 array, object, callable, iterable이 compound타입에 해당한다.
나머지 2개는 special 타입으로 resource, NULL이 special 타입에 해당한다.

integer는 정수를 표현하는 데이터 타입으로 자연수와 0, 음수가 이에 해당한다.

bonus
  • echo : echo는 php에서 데이터의 형식을 막론하고 화면에 무엇인가를 표현하는 방식이다.
  • print() : echo와 같은 역할을하며 ()안의 값을 html로 출력한다.
  • 세미콜론(;) : php는 반드시 하나의 구문(or 명령)이 끝나면 세미콜론을 기입하여 구문의 끝을 알린다.
  • 표현식 : 컴퓨터 언어가 실행된 후에 최종적으로 어떠한 값으로 바뀌는 것들을 표현식이라고 한다.
