PHP?

php라는 언어의 활용을 본격적으로 공부하기 전에 php가 무엇이며 어떤 역할을 하는 확인해보고자 합니다.
(다음의 내용은 'https://www.w3schools.com/php/php_intro.asp'를 참고하였습니다.)

  1. php는 서버측에서 작동하는 스크립트언어입니다.
  2. php 파일(.php)은 텍스트와 HTML, CSS, JavaScript, PHP 코드를 수용할 수 있습니다.
  3. php 코드는 서버에서 작동하며 작업의 결과를 순수한 HTML로 브라우저에게 리턴합니다.
  4. php 동적페이지 컨텐츠를 생성할 수 있습니다.
  5. php는 서버에 있는 파일을 생성하고 열고, 읽고, 쓰고, 삭제하고, 닫을 수 있습니다.
  6. php는 양식데이터를 수집할 수 있습니다.
  7. php는 쿠키를 보내고 받을 수 있습니다.
  8. php는 데이터베이스에 있는 데이터를 추가하고, 삭제하고, 수정할 수 있습니다.
  9. php는 유저의 접근을 컨트롤하는것에 사용될 수 있습니다.
  10. php는 데이터를 암호화 할 수 있습니다.

w3s에서 언급한 내용을 통해 Java로 웹을 구현할 때 사용했던 jsp/servlet의 역할을 하고 있다는 것을 알게되었습니다.

Mechanism of PHP

php가 클라이언트의 web browser와 server computer의 관계에서 어떻게 동작하는지 php의 작동 원리입니다.
(유튜브 생활코딩의 이고잉 선생님이 함께해주셨습니다. 선생님은 앞으로도 많이 함께해주실 예정.. 감사합니다..)

클라이언트가 서버 컴퓨터에 '.html'의 확장자를 가진 요청을 하였을 경우
  1. 클라이언트의 웹 브라우저가 서버컴퓨터에 'index.html'의 요청을 합니다.
  2. 서버컴퓨터의 웹 서버 소프트웨어는 '.html'의 확장자를 확인하고, 웹 서버 스스로 처리할 수 있는 요청임을 인지합니다.
  3. 웹 서버는 서버컴퓨터의 저장소에 설치되어있는 htdocs에서 index.html파일을 읽어 클라이언트에게 전송합니다.
  4. 클라이언트의 웹 브라우저는 전송받은 html을 해석해서 화면에 출력합니다.
  5. '.html'의 데이터는 정적입니다.
클라이언트가 서버컴퓨터에 '.php'의 확장자를 가진 요청을 하였을 경우
  1. 클라이언트의 웹 브라우저가 서버 컴퓨터에 'index.php'의 요청을합니다.
  2. 웹 서버는 '.php'의 확장자를 확인하고 스스로 처리할 수 없는 요청임을 인지합니다.
  3. 웹 서버는 '.php' 요청의 처리를 PHP 프로그램에 위임합니다.
  4. PHP는 htdocs라는 디렉토리 안에 있는 'index.php' 파일을 확인합니다.
  5. 해당 파일에 적혀있는 php 언어를 해석하고 최종적으로 '.html' 파일을 생산합니다.
  6. PHP에 의해 생산된 ',html'을 웹 서버가 클라이언트에게 전송합니다.
  7. 클라이언트의 웹 브라우저는 전송받은 html을 해석해서 화면에 출력합니다.
  8. '.php'의 데이터는 '' 안에 적혀있는 php 문법에 따라 실시간으로 html을 생성하므로 동적입니다.

Data Type of PHP

프로그래밍 언어를 사용하는 이유 중에 하나는 데이터를 어떠한 의도에 따라 처리하기 위함일 것입니다. 이러한 이유로 인해 '프로그래밍 언어가 어떠한 데이터 타입을 제공하는가 그리고 각각의 데이터 타입을 처리하는 방법을 어떻게 제공하는가'에 대해 공부하는 것은 필수적입니다.

'php.net'의 php manual에 따르면 php는 10개의 기본 데이터 타입을 제공합니다.

그중에 4개는 scalar 타입입니다.

  1. bool
  2. int
  3. float(floating-point number, aka double)
  4. string

(네이버 사전에 따르면 'scalar'가 '수치(또는 실수)로 표시되는 수량'의 의미를 가지고 있다고 하는데... string은 왜 scalar 타입인걸까?)

또 다른 4개는 compound 타입입니다.

  1. array
  2. object
  3. callable
  4. iterable

나머지 2개는 special 타입에 해당합니다.

  1. resource
  2. NULL

String

String은 문자를 표현하는 데이터타입입니다.

Array

배열은 다른 데이터를 정리하여 보관하는 데이터타입입니다.

배열은 'array(element1, element2, ...)'와 같은 형태로 생성합니다.
'$values = array(element1, element2, ...)'와 같이 생성할 때 변수에 대입하여 사용합니다.

배열은 인덱스를 가지고있으며 가장 첫번째 엘리먼트의 인덱스는 0입니다.
'$values[0]'은 element1을 호출합니다.

배열 생성 후 배열의 가장 뒤쪽에 새로운 요소를 대입하고자 할 때에는 'array_push()'를 사용합니다.
'array_push(배열명(변수명), 대입할 값)'과 같이 선언할 수 있습니다.

URL Parameter

입력에 따른 동작을 실행하고 결과를 출력하는 것은 모든 소프트웨어의 기본 형태입니다. php는 웹 어플리케이션을 위해 만들어진 언어 입니다. 웹 어플리케이션에서는 어떠한 입력이 있을 수 있고 그에 따른 출력은 무엇인지 알아보겠습니다.

.php에는 URL을 통한 입력을 할 수 있습니다. "FileName.php?parameterName1=value1¶meterName2=value2"라는 URL 보겠습니다. 이 URL의 의미를 풀어보면 'FileName'이라는 이름의 php 어플리케이션에 'parameterName1은 value1이다.'라는 입력값과, 'parameterName2는 value2d이다.'라는 입력값을 전달하는 것을 의미합니다. 이것은 하나의 php 어플리케이션이 전달 받는 입력값에 따라 무한히 많은 웹 페이지를 생성할 수 있음을 의미합니다.
